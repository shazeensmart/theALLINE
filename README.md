# TheALLINE

TheALLINE is a shopping comparison website that helps users find the best deals across multiple online stores.

## Features

- Product comparison (Amazon, Flipkart)
- Buy links from different shopping platforms
- YouTube product review videos
- Shopping cart
- Blog for latest deals and offers
- Mobile friendly design
- Live chat support

## Purpose

The goal of TheALLINE is to allow customers to:

- Compare product prices
- Watch product reviews
- Choose the best online store
- Buy products easily from one platform

## Technology Used

- HTML
- CSS
- JavaScript
- GitHub Pages (free hosting)

## Future Plans

- Product search engine
- Automatic price comparison
- User accounts
- Admin dashboard
- Android mobile app
